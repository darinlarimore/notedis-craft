# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-22

### Added
- Initial release
- Automatic widget injection on all pages
- Plugin settings page in Craft CMS control panel
- Site key configuration (required)
- API endpoint configuration
- Button position selection (bottom-right, bottom-left, top-right, top-left)
- Button color customization
- Logged-in users only option
- Show in control panel option
- Widget source selection (local or CDN)
- Support for Craft CMS 4.x and 5.x
- PHP 8.0.2+ compatibility
- Proper input validation and sanitization
- Privacy notice in settings
- Comprehensive documentation

### Security
- Follows Craft CMS security best practices
- All settings validated and sanitized
- All output properly escaped
- Secure storage of site keys in project config
